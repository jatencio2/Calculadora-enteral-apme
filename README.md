## Descripción
Herramienta web de soporte a la decisión clínica para la prescripción 
de nutrición enteral y suplementación oral. Incluye base de datos APME 
con 77 fórmulas comerciales disponibles en Colombia.

## Características
- 77 fórmulas verificadas (22 polvos · 19 líquidas sonda · 36 líquidas oral)
- Cálculo de Requerimiento Calórico Estimado (RCE) por edad y peso
- Deuda calórica ajustable (5–100% del VCT)
- Filtros por contexto nutricional (4) y patológico (23)
- Vía oral y sonda nasogástrica con prescripción por bolo
- Presentación requerida día/mes con algoritmo ⌈ceil⌉ corregido
- Despliegue: Google Apps Script Web App
